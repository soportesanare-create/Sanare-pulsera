# Sanaré – Monitor de Infusión (Wearable)

Interfaz HTML (tema claro) para visualizar señales de un wearable en sala de infusión.

## Uso
1. Abre `index.html` localmente, o
2. Sube a GitHub y activa **GitHub Pages** (Settings → Pages → Deploy from branch).

## Modo demo
Pulsa **Iniciar demo** para ver datos simulados y alertas.

## Datos en vivo
En `index.html`, configura `WS_URL` con tu WebSocket (`wss://...`) y envía JSON con el formato indicado en el código.
